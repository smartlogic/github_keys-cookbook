Description
===========
Installs/Configures a server user with keys based on members of a github 
organization

Requirements
============
None

Attributes
==========

    "github_keys" : {
      "force_key_load" : false,
      "username" : false,
      "github_org" : false,
      "create_user" : true
    }

Usage
=====

Add `github_keys` to your run list
